* **`ft_putnbr`** : Affiche un nombre entier sur la sortie standard.

## 🇬🇧 English
This repository contains the functions I developed during my **March 2026 Piscine**. The goal was to recode standard library functions and master C programming fundamentals.

### 🧩 Implemented Functions:
* **`ft_atoi`** : Converts a string to an integer (int).
* **`ft_split`** : Splits a string into an array of strings based on a delimiter.
* **`ft_fibonacci`** : Calculates the n-th term of the Fibonacci sequence (recursion).
* **`ft_putnbr`** : Displays an integer on the standard output.
* **`ft_rev_int_tab`** : Reverses the order of elements in an integer array.
* **`ft_sort_descending`** : Sorts an integer array in descending order.
* **`ft_fizzbuzz`** : The classic loop and conditional logic exercise.
